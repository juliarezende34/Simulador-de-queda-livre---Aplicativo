# Simulador de queda livre: Aplicativo

## Instruções de download
- Ao abrir o repositório do GitHub, clique no botão verde "Code" e baixe o .zip do projeto;
- Descompacte a pasta na pasta desejada no seu computador;
- Abra o arquivo "New Unity Project.exe", que a simulação será executada.

## Como simular?
- Bloco da esquerda: não sofre força de arrasto, podendo ter sua massa e altura inicial inseridas pelo usuário;
- Bloco da direita: pode sofrer força de arrasto caso K (constante de resistência do ar) seja maior que 0. Também pode ter sua massa e altura inicial alteradas pelo usuário;
- Digite os valores desejados e aperte o botão "Start";
- Após a queda, aperte "Reset" para uma nova simulação;
- Use o botão "Sair" para fechar a simulação.
